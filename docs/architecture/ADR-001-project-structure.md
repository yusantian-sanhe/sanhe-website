# ADR-001 Project Structure

## Status

Accepted

---

## Date

2026

---

## Context

Project Harvest is being developed as a production-ready international B2B agricultural export website.

The project is expected to continue growing with:

- Homepage
- Products
- About
- Contact
- CMS integration
- SEO
- Blog
- Future admin system

A clear architecture is required to ensure long-term maintainability.

---

## Decision

The project adopts the following principles:

- App Router (Next.js)
- TypeScript
- Tailwind CSS
- next-intl
- Feature-oriented components
- Page-based internationalization
- Reusable UI components
- Strict folder organization

---

## Consequences

Benefits:

- Easy maintenance
- Easy expansion
- Enterprise-ready structure
- Suitable for teamwork
- Easy CMS integration
- Clear responsibility separation

Future architecture decisions will continue to be recorded in the ADR directory.
